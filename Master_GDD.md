# Master GDD: Fit & Mood (MoodFit)

> **슬로건**: "내 몸의 수치부터 마음의 기분까지, 당신을 위한 오늘의 맞춤 솔루션"

---

## 1. 서비스 핵심 가치 (Core Values)

| 가치 | 설명 |
|:---:|:---|
| **개인화** | 단순 몸무게가 아닌 MBTI, 현재 기분, 타겟 부위를 고려한 초정밀 큐레이션 |
| **효율성** | 서버 비용 최소화를 위해 로컬 데이터를 우선 활용하되, AI API로 확장성을 열어둔 하이브리드 구조 |
| **통합** | 다이어트(식단/운동)와 스타일링(코디)을 하나의 흐름으로 연결 |

---

## 2. 앱 주요 구조 (App Architecture)

```
[Input Layer] → [Content Engine] → [UI Screens]
```

### A. 정보 입력 계층 (Input Layer)

- **신체 데이터 (고정)**: 성별, 키, 현재 체중, 목표 체중, 살 빼고 싶은 부위
- **성향 데이터 (준고정)**: MBTI, 선호하는 맛, 체형(골격 유형)
- **상태 데이터 (가변)**: 현재 기분(감정), 계절/날씨

### B. 콘텐츠 엔진 (Engine - Hybrid)

- **Local DB**: 미리 준비된 표준 레시피, 운동법, 체형별 코디 정석 (태그 기반 매칭)
- **AI API**: 특수한 상황(냉장고 파먹기, 정밀 분석) 또는 사용자가 Refresh 버튼을 눌렀을 때 호출

## 3. 화면별 구성 및 기능 (Screen Flow)

### Screen 1: 홈 대시보드 (`Dashboard.tsx`)

- 기분 선택기: 상단에 "오늘의 기분은?" (아이콘: 😊, 😔, 😡 등)
- 오늘의 한마디: 기분과 MBTI를 조합한 AI 생성형 격려 메시지
- 퀵 추천: 현재 날씨와 내 체형에 맞는 **'오늘의 베스트 코디'**와 **'오늘의 추천 한 끼'** 카드

### Screen 2: 다이어트 & 레시피 (`DietRecipe.tsx`)

- 잡지 스타일의 고화질 음식 이미지와 레시피 가이드
- 재료 정보: 식재료의 이름, 유래, 영양 성분 카드뉴스
- 레시피: 로컬 DB 기반 요리법 + AI 모드 시 '냉장고 파먹기' 기능
- 칼로리 계산기: 일일 목표량 대비 잔여량 시각화 *(미구현)*

### Screen 3: 스타일링 & 운동 (`StyleWorkout.tsx`)

- 부위별 코디법: '살 빼고 싶은 부위'를 보완하는 체형별 의상 매칭
- 부위별 운동: 선택한 부위 집중 운동법 이미지 순서 제공
- 계절 필터: 현재 계절에 맞는 소재와 색상 조합 제안
- AI 리포트 다운로드: 고화질 이미지 포함 `.html` 파일 다운로드

### Screen 4: 프로필 설정 (`ProfileSetup.tsx`)

- 신체 정보 입력 (성별, 키, 몸무게, 체형)
- MBTI 선택, 타겟 부위 선택
- AI 엔진 ON/OFF 토글, AI 프로바이더 선택 (Gemini / OpenAI)
- 언어 설정 (한국어 / English)

---

## 4. 데이터 모델 (Data Model)

### 핵심 타입 (`types.ts`)

- `UserProfile`: 사용자 전체 프로필 (기분, 체형, MBTI, AI 설정 등)
- `RecommendationResult`: AI/로컬 추천 결과 (quote, recipe, outfit, workout)
- `Recipe`, `Outfit`, `Workout`: 각 콘텐츠 타입 (모두 `TaggedItem` 상속)

---

## 5. 향후 구현 시 고려사항 (Roadmap)

### 미구현 기능

- [ ] **칼로리 트래커**: 실제 기록 기능과 연동
- [ ] **운동 타이머**: 운동 가이드 화면에 카운트다운 타이머

### 데이터 태그 시스템

모든 로컬 데이터(레시피, 코디 등)는 `sad`, `summer`, `ESTJ`, `arm_fat` 같은 태그를 가지며, AI API 역시 이 태그를 반환하도록 프롬프트를 설계해야 합니다.

### 스위치 로직

`use_ai_api = true/false` 변수 하나로 전체 데이터 소스를 제어할 수 있게 구성합니다.
